Wage Gap Analytics

This webstie is part of a school project, it offers data about wage in a plethera of countries.

Project background:

The project is part of "data science" class for 10th grade students of Tichonet's High School in Tel Aviv.
The project combines two main areas of knowledge:
- Programming in a web environment
- Data science
As part of the project, I located databases on the network and intend to build around them a website that allows access to the databases, data analysis and presentation to users in a convenient way.
The site will allow the user to explore the data in the chosen field and try to reach valuable insights and conclusions.
